# Homebrewery Clone
This is a simplified and lightweight webpage based on NaturalCrit's Homebrewery, accessible [here](https://github.com/naturalcrit/homebrewery).

## Markdown-based Editor
You can freely use markdown, a widely used, lightweight markup language, to write your documents. Type in the text box to the left and watch as the changes appear on screen in real time.

## CSS Customization
Using the buttons at the top of the text box, you may switch between markdown text and the CSS tab. Every element's appearance can be changed with custom rules.

### Inline CSS
Using curly brackets {}, you may apply direct CSS styling to the element before them.
Like this.
{color: blue;}

### Custom-Class Blocks
Double curly brackets immediately followed by a class name will apply that class's styling to the content inside.

{{text-box
##### You may do it like so!
This is a custom text box. Its CSS rules can be seen in the *Custom CSS* tab.

Check the text-box to see how the code looks!
}}

\column

## Column Breaks
Column breaks can be triggered manually by typing `\ column` on an empty line (without the space).

## Saving and Loading Your Work
Using the two buttons at the bottom right of the page, you can:
* Save your work using the **Floppy Disk**
* Load a local file using the **Eject Button**

![](https://i.postimg.cc/pTZHCsD3/image.png)
{display: flex; justify-content: center; margin-bottom: 10px;}

This editor only supports .txt files that follow a specific format. Press the Save Button and have a read through the file to see how it looks.

*You may also use CTRL + S to quickly save your work.*

{{wide
| 1d6  | Random Thing | Other Random Thing |
| :--: | :----------: | :----------------: |
| 1-2  | A thing      | Other thing        |
| 3-4  | Thing thing  | Other other        |
| 5-6  | Some thing   | Some other         |
}}

## Other Custom Classes
Writing ```wide``` between curly brackets will force the element to take the entirety of the horizontal space available. It works well with headings and tables -- also notice how the table above is written in pure markdown.

\column

## Other page formats
Other page sizes are only partially supported at the moment. You can switch to A5 by changing the ```.page``` width and height to 148mm and 210mm respectively.

**Changing page size on a pre-written document of another size (like this one) might cause crashes. Always start a fresh document.**

{{pageNumber 1}}

\page

# Page Breaks
Page breaks can be triggered manually by typing `\ page` on an empty line (without the space).

## And If You Need To Customize a Page...
We got you covered. Check the last lines of Custom CSS tab.

You can use the #page-x id to make changes to a specific page only. 

\column

## Page Numbers
Can be added using the `pageNumber` class between curly brackets. It can be set to **auto**, a **number** of your choice, or `auto - offset` (e.g. `auto - 5`).

{{pageNumber auto}}
