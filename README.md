# Homebrewery HTML

This is a lightweight simple document creator based on [NaturalCrit's Homebrewery](https://github.com/naturalcrit/homebrewery), provided under the MIT License.

## Usage

Simply load the index.html file in your browser and start editing the content right away.

### Saving Content

Can be done using the 'Floppy Disk' button on the bottom right of the screen, or by using the CTRL + S combination.

This will prompt the user to download a .txt file containing raw text and custom CSS values (created by the user).

### Loading Content

Can be done using the 'Eject' button on the bottom right of the screen.

This will prompt the user to upload a .txt file that uses the same format as above. If the format is not followed, it will result in possible errors and issues.

### Printing Content

Can be done with the browser's default printing function (or by pressing CTRL + P). By default, it will only print the pages of the document as seen in the preview window (right side of the webpage).
